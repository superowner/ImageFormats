imageformats
============

.NET modules for decoding obscure graphics formats:

- .PPM, .PGM, .PBM (Netpbm images)
- .TGA (Targa)
- .PCX (ZSoft Paintbrush)
- .IFF (Amiga ILBM images)
- .SGI, .RGB, .BW (SGI images)
- .RAS (Sun raster images)
- .MAC (MacPaint)
- .CUT (Dr. Halo)
- .XPM (X Window PixMap)
- .DICOM (experimental support)
